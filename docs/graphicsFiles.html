Saving your map as a graphics file

The best way to save your map as a graphics file is by saving it as a PDF file (from the File menu), and then edit it in Adobe Illustrator or a similar program that deals with full vector graphics. If you do this, the symbols will appear in high resolution, much higher than they appear on screen. Save PDF by choosing Save Tree as PDF from the File menu.

Make sure that your base map is of high enough revolution for your publication needs.
